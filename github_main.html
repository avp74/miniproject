Using Github

  • How to make a repository

    • To make a repository you need to first create a GitHub account. After creating an account, on the top right hand corner, next to your profile icon, you will see a “+" icon, with a drop down menu. The first option is “New Repository", by clicking on it, you will be transferred to a new page titled, “Create a New Repository”. Here you will create a Repository name, and other options. By clicking the green button, “Create Repository”, your Repository will be ready to use.

  • How to make a commit in WebStorm

    • After your github setup has been completed in WebStorm, go to the VCS tab, and you will see the commit button with a green check mark next to it. It will lead you to a pop up called “Commit Changes” where you can select the files you wish to commit and add a commit message of what your task or edit was based on. After you have done that, you can press the blue commit button on the bottom right corner.

  • How to make a branch

    • You can create a branch in Webstorm by clicking on the bottom right corner labeled “Git: master”, a pop up menu will give you the option to create a new branch called: “+ New Branch”. After clicking on it, you will proceed to create a new branch by giving it a unique name and then click the blue button labeled “Create”. A green popup message will appear once it is created to confirm the request.

  • How to merge one branch to another

    • To merge one branch to another, you click on “Git: (your branch name)” on the bottom right corner and “checkout” the base branch you want it to merge your changes into. Click on the branch you wan want the changes to be merged into again and choose the "Merge into Current" option from the submenu. Or you can click the top “VCS” tab, choose “Update Project”, and your current branch will be updated with the branch you were working on.

  • How to make a Pull Request and merge on Github

    • You make a pull request on Github by clicking the “Pull Requests” option in your specified repository page where your project is located. Click on the pull request you'd like to merge and you will be directed to a new page to confirm the merge of the commits into the base branch.
