Webstorm Terms

  • Terminal

    • The terminal emulator works with the command line, it is used for entering commands and running with your default system shell. The terminal emulator runs with the current directory set to the root directory of the current project.

  • Login

    • When you log into webstorm it will instruct you to sign in, enter your username and password and you can create a new project

  • File

    • n the Project Tool Window, select the directory where you want to create a file, press N/A, and then select File from the list. In the New File dialog that opens, type the filename and extension. You can type the whole directory structure before the new filename. If the nested directories do not yet exist, they will be created.

  • Folder / Directory

    • Folders/Directories are where your files go, they make it easy to organize your files.

  • Webstorm

    • WebStorm is a powerful IDE for modern JavaScript development. WebStorm provides full support for JavaScript, TypeScript, HTML, CSS as well as for frameworks such as React, Angular, and Vue. js right out of the box, no additional plugins are required.

  • Integrated Development Environment (IDE)

    • is a software application that provides comprehensive facilities to computer programmers for software development

  • HTML5 Boilerplate

    • A is an HTML, CSS and JavaScript front-end template that helps you build a fast and adaptable web app or site.

  • Grunt

    • WebStorm integrates with the Grunt JavaScript Task Runner. WebStorm parses Gruntfile.js files, recognizing definitions of tasks and targets, shows tasks and targets in a tree view, lets you navigate between a task or a target in the tree and its definition in the Gruntfile.js file, and supports running and debugging tasks and targets

  • HTTP

    • HyperText Transfer Protocol is an application-layer protocol used to transfer data from a web server to a web browser to view web pages.

  • Web browser

    • A software application for accessing information on the World Wide Web that retrieves content from a web server to then display on the web page.

  • HTML

    • Hypertext Markup Language is the standard markup language for documents that defines the meaning and structure of web content in a web browser.

  • CSS

    • Cascading Style Sheets is a style sheet language used to design and layout the presentation of a document written by HTML.
